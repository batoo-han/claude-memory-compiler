## Raw Content

### Контекст заказа
Клиентка Ирэна Иванова заказала доработку существующего конструктора резюме cv.ibestresume.ru. Сайт уже работает, есть базовая функциональность: регистрация, заполнение резюме, превью, экспорт PDF, выбор шаблона.

### Что уже есть на сайте (подтверждено исследованием)
- React 18.3 SPA через Vite, Bootstrap 5
- Кастомный datepicker/yearpicker, Nunjucks (шаблоны), react-tel-input
- REST API v1, cookie-auth с token-refresh
- CRUD резюме, загрузка аватара, превью в реальном времени
- Админка: управление пользователями, шаблонами, подсказками, биллингом

### 12 категорий доработок
1. Регистрация и аккаунт (восстановление пароля, email-подтверждение, автосохранение)
2. Даты (месяц+год вместо полной даты, ручной ввод)
3. Образование (расширение типов, гибкий выбор вуза)
4. Опыт работы (выпадающий список отраслей)
5. Навыки и достижения (hard/soft skills, фикс багов)
6. Фото/аватар (встроенный редактор: кадрирование, поворот, масштаб)
7. Интерфейс (rich-редактор, подсказки эксперта, загрузка из Word)
8. ИИ-функционал (YandexGPT: автодополнение, проверка резюме, генерация достижений)
9. Экспорт и совместимость (кодировка PDF, шаринг по ссылке)
10. Дополнительные блоки (тренинги/курсы, конструктор сопроводительного, портфолио)
11. Баги (выход из аккаунта по «Назад», сброс счётчика навыков)
12. Разное (ограничение длины, шрифты, пояснения)
